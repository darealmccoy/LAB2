Course Catalog Description

This course will cover the creation of webpages and websites using HTML, CSS, Javascript and graphical applications. Networked multimedia distribution technologies will be explored as well. The design of effective Web site including page layout, user interface design, graphic design, content flow and site structure as well as management and design consideration are addressed. Students will design and create a major Web Site with multiple pages and cross-linked structures. Prerequisites: [none] Credit: 2-2-3 Semester Hours

Course Outcomes:

Students completing this course will be able to:

  • Recognize HTML, CSS and Javacript mark-up and code in a web page/application
  • Select the proper mark-up tags or code to achieve a particular result.
  • Identify impoperly used mark-up code.
  • Produce modern standards complient web pages.
  • Deploy web pages to a public server.

Course Objectives:

At the conclusion of this course, each successful student will be abile to:

  • Explain client and server architecture of the Internet and related web technologies.
  • Use a basic text editor and other software tools to create web page using HTML, CXS, Javacript, JQuery, and images.
  • Deploy web pages to a Linux-based web server using stfp.
